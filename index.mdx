## Overview

TaskFlow is a lightweight REST API for managing projects and tasks.

<Card title="Projects" icon="pen-to-square" horizontal>
  Create projects to organize related tasks.
</Card>

<Card title="Tasks" icon="check" horizontal>
  Create, update, retrieve, and delete tasks.
</Card>

This API follows a simple workflow:

1. Create a project
2. Create tasks inside the project
3. Retrieve projects tasks
4. Update task status
5. Delete tasks

## Authentication

All API requests must include a bearer token in the Authorization header.

**Example:**

```json
curl https://test-writechoice.onrender.com/projects \
  -H "Authorization: Bearer YOUR_API_KEY"
```
